#+title: A TEI XML file for the Kokinwakashu combined from lexically annotated sources and bibliographically annotated sources

This repository combines two sources of TEI XML files for the Kokinwakashu, a classical Japanese poetry anthology.

The first source is a lexically annotated TEI XML file, which is a subset of the [[https://github.com/yamagen/hachidaishu][Hachidaishu Vocabulary Dateset]].

The second source is a bibliographically annotated TEI XML file, which is a digitalized version of the Karoku 2 Manuscript of the Kokinwakashu, provided by the [[https://github.com/kokubunken/kokinwakashu][嘉禄二年本『古今和歌集』の翻刻およびTEI/XMLデータ]] repository.

Note: The process of combining is assisted by Large Language Models.
